Cato, the cat

A casual match-3 game where players help Cato, meet his needs by matching pieces representing hunger, thirst, and fun.

The game features dynamic animations based on Cato's status, with a leaderboard to engage competitive players.

Release date: 2022

Development Process

The project was created to improve our pixel art skills, working closely with a pixel artist to refine techniques and implement them into the game.


My responsibilities included:

  • Game design, including the creation and documentation of the core gameplay and key mechanics.
  • Programming the match-3 mechanics, identifying matching pieces and their combos.
  • Implementing dynamic changes in the character's animations to reflect his needs.
  • Actively contributing to the definition of the pixel art style.
  • Developing a leaderboard integrated with the Steam API to track player scores.
  • Publishing the game on Steam.
